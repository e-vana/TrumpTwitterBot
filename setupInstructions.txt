// Installations
Download and install node.js

// How to setup folder structure and start serve
Put all of the files in this folder into a folder on your local machine.

// EXAMPLE
You placed the contents of this folder into C:/NodeProjects/

// Type these 
WINDOWS + R, type CMD
cd C:/
cd NodeProjects

// Type this to install all node dependencies, might take a while
npm install

// type this to start the development server with hot reload
nodemon
